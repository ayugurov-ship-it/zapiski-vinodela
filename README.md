# 🍷 Записки винодела

Автоматическая публикация постов в Telegram-канал о домашнем виноделии.

## Темы постов

- Процесс изготовления вина (рецепты, этапы)
- Сорта винограда и фруктов для виноделия
- Лайфхаки домашнего винодела
- Хранение вина
- Сочетание вина с едой
- Подача и сервировка
- Крепость и содержание сахара
- Обзоры полезных товаров (Ozon, Wildberries)

## Настройка

### 1. Создать Telegram-бота

1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь `/newbot` и следуй инструкциям
3. Сохрани полученный токен бота
4. Добавь бота администратором в канал "Записки винодела"

### 2. Узнать Chat ID канала

1. Добавь бота в канал как администратора
2. Отправь любое сообщение в канал
3. Открой `https://api.telegram.org/bot<TOKEN>/getUpdates`
4. Найди `chat.id` канала (обычно начинается с `-100`)

### 3. Настроить GitHub Secrets

В настройках репозитория (Settings → Secrets → Actions) добавь:

- `TELEGRAM_BOT_TOKEN` — токен бота
- `TELEGRAM_CHAT_ID` — ID канала

### 4. Публикация

Посты публикуются автоматически каждый день в 10:00 MSK через GitHub Actions.

Также можно запустить вручную: Actions → Publish Post → Run workflow.

## Структура

```
posts/           — посты в формате Markdown
scripts/         — скрипты публикации
.github/workflows/ — автоматизация
```

## Добавление постов

Создай файл в папке `posts/` в формате:

```
NNN_название.md
```

Формат файла:

```markdown
---
title: Заголовок поста
tags: #вино #рецепт
---

Текст поста...
```
